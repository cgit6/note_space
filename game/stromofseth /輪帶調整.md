   這部分統計的是每個玩家的贏得金額分佈，按“bucket”（金額區間）分類，並統計玩家在每個桶中“擊中”（即贏得落入該區間）的次數。數據基於 10,000 個玩家，每個玩家 4,500 局的模擬。
   `[0,0]`：贏得 0（完全沒贏，輸掉投注）。
   `(0,1)`：贏得 0 到 1 倍（小贏，但不足以覆蓋投注）。
   `[1,2)`：贏得 1 到 2 倍（小贏）。
   ...
   `[10000,+inf)`：贏得 10,000 倍以上（大獎，極高倍數）。

   擊中次數的解釋
   0x：玩家在該桶中 0 次擊中（從未贏得落入此區間的金額）。
   1x：擊中 1 次。
   2x：擊中 2 次。
   3+x：擊中 3 次或更多（表示多次擊中，顯示該桶是常見贏得區間）。
   為什麼重要：這顯示玩家體驗的頻率分佈。例如，如果一個桶的 3+x 高，表示大多數玩家多次遇到該贏得水平。

   以下按桶順序分析關鍵數據（百分比為玩家比例，CI 顯示統計可靠性）：

   `[0,0]`（完全沒贏）：
   0x: 10.96% → 只有 11% 玩家從未完全沒贏。
   1x: 9.84% → 10% 玩家只沒贏 1 次。
   2x: 9.46% → 9% 玩家沒贏 2 次。
   3+x: 69.74% → 69.74% 玩家多次（3+ 次）完全沒贏，表示大多數玩家經常輸掉投注。這是低桶集中點，反映遊戲的“輸多贏少”本質。
   `(0,1)`（小輸，贏 0-1 倍）：
   0x: 100.00% → 100% 玩家從未擊中此桶，幾乎沒有人只贏一點點就停。
   1x/2x/3+x: 0% → 此區間極少出現，玩家要麼輸光，要麼贏更多。
   `[1,2)`（小贏，贏 1-2 倍）：
   0x: 95.01% → 95% 玩家從未贏 1-2 倍。
   1x: 4.77% → 5% 玩家只贏 1 次。
   2x: 0.22% → 很少玩家贏 2 次。
   3+x: 0% → 幾乎沒有人多次小贏。整體顯示小贏罕見，玩家傾向大贏或大輸。
   `[2,5)`（中等贏，贏 2-5 倍）：
   0x: 81.71% → 82% 玩家從未贏此區間。
   1x: 15.11% → 15% 玩家贏 1 次。
   2x: 2.66% → 3% 玩家贏 2 次。
   3+x: 0.52% → 很少多次。顯示中等贏較常見，但不頻繁。
   `[5,10)` 和 `[10,20)`（中等到大贏）：
   這些桶的 0x 下降（77.82% 和 71.00%），1x 上升（17.70% 和 21.45%），表示贏得機會增加，但仍非主流。3+x 低，顯示不常重複。
   `[20,50)` 和 `[50,100)`（大贏）：
   0x: 63.61% 和 70.92% → 更多玩家從未大贏。
   1x: 24.05% 和 21.33% → 約 20-24% 玩家贏 1 次。
   2x 和 3+x 略增，但仍低。顯示大贏偶爾發生，但不穩定。
   `[100,300)`（超大贏）：
   0x: 61.75% → 62% 玩家從未贏 100+ 倍。
   1x: 26.14% → 26% 玩家贏 1 次。
   2x: 10.32% → 10% 玩家贏 2 次。
   3+x: 1.79% → 少量多次。顯示此桶較常見，玩家有機會多次大贏。
   `[300,500)`、`[500,1000)`、`[1000,2000)`、`[2000,10000)`（極大贏）：
   這些高桶的 0x 高（77-88%），1x 低（11-22%），2x 和 3+x 幾乎 0%。表示極大獎極稀有，大多數玩家從未擊中，即使擊中也只 1 次。
   `[10000,+inf)`（巨獎，10,000 倍以上）：
   0x: 88.08% → 88% 玩家從未贏巨獎。
   1x: 11.92% → 12% 玩家贏 1 次。
   2x/3+x: 0% → 從未重複。顯示巨獎極端稀有，是“全或無”的象徵。

   `0x、1x、2x、3+x` 是統計每個玩家在這個“盒子”（桶）裡“中獎”（贏到該金額範圍）的次數。用白話說：
   0x：從來沒中過這個盒子（玩家在所有 4,500 次遊戲中，沒一次贏到這個範圍的錢）。
   1x：只中過 1 次（玩家只在 1 次遊戲中贏到這個範圍）。
   2x：中過 2 次（玩家在 2 次遊戲中贏到這個範圍）。
   3+x：中過 3 次或更多次（玩家在 3 次以上遊戲中贏到這個範圍）。
